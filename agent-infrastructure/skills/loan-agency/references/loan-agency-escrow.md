# Loan Agency Escrow Operations

## Overview

This skill covers escrow agreement negotiation, funding, release, and the internal guardrails that govern the escrow agent role. Escrow is a specialized subset of deals — not all loan agency deals involve escrow.

**Key principle:** The escrow agent acts strictly as a neutral party — holding and releasing funds (or documents) per the agreement's explicit terms. The agent does not exercise discretion; it follows instructions or refuses to act.

---

## Critical Rules (Read First)

### 1. Escrow Guardrails Require Senior Approval to Override

Internal guardrails set minimum requirements for escrow agreements. Provisions outside guardrails require **senior leadership approval** before execution.

### 2. Releases to Non-Parties Require Escalation

Releases default to **parties to the agreement** or lenders/borrowers under the related credit agreement. Releases to non-parties → escalate to senior leadership / Legal.

### 3. Joint Written Instructions Are the Default

Releases should require **joint written instructions** from relevant parties (not unilateral instruction).

## 2. Negotiation Guardrails

### Required Provisions

| Provision | Requirement |
|-----------|-------------|
| Wire instructions | In the agreement itself |
| Release instructions | Joint written instructions; specify who and in what form |
| Release recipients | Default to agreement parties or credit agreement lenders/borrowers |
| Dispute resolution | Refuse-to-act provision for conflicting claims |
| Exculpation | General and special — not liable for good faith actions |
| Indemnification | Agent indemnified by parties |
| Fee/set-off | Agent entitled to fees and set-off against escrowed funds |

### Escalation Triggers

| Provision | Escalation |
|-----------|-----------|
| Release to non-parties | Senior leadership / Legal |
| Unilateral release authority | Senior leadership |
| Missing dispute resolution | Senior leadership |
| Missing/limited exculpation or indemnification | Senior leadership |
| Any provision outside guardrails | Senior leadership |

---

## 3. Escrow Funding (Cash Escrow)

1. Set up sub-account in escrow banking platform
2. Provide wire instructions to depositing party (must match agreement)
3. Receive and confirm funds
4. Send written confirmation to all parties
5. Segregate agent fees if agreement provides

---

## 4. Release with Instructions

1. Receive joint written instructions from authorized parties
2. **Verify:** signatures authorized, form matches agreement, conditions satisfied
3. **Verify recipient:** party to agreement → proceed; non-party → escalate
4. Process release (wire or document delivery)
5. Confirm to all parties
6. Update records

---

## 5. Release without Instructions (Expiry/Default)

For agreements with automatic release provisions:
1. Identify trigger (expiry date, condition satisfied, default)
2. Follow exact agreement language
3. Notify all parties of pending release
4. Process per agreement terms
5. If any party objects: invoke refuse-to-act provision

---

## 6. Dispute Resolution

1. **Do NOT release** — hold pending resolution
2. Invoke refuse-to-act provision
3. Options: parties resolve jointly, interpleader, arbitration/mediation
4. Agent fees continue to accrue
5. Coordinate with legal counsel if interpleader required

---

## 7. Escrow Termination

1. All amounts released / documents delivered
2. Confirm zero balance
3. Close sub-account
4. Send termination notice
5. Final billing

**Note:** Some escrows outlive the related credit facility. Continue monitoring until the escrow's own termination conditions are met.
