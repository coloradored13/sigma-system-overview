### 2. Remittance Pricing Trends

- By provider type ($200): Banks ~12.66%, MTOs ~5.35%, Mobile carriers ~3.87%

### 4. Economics of Stablecoin-Based Remittances
**Traditional Full Chain:** Service fee 2-6% + FX markup 1-3% = 4-8% typical; banks 12.66% for $200
**Stablecoin Full Chain:** On-ramp 0-4.5% + blockchain <$0.50 + off-ramp 0.5-2.0% + FX 0-0.5% = 1-3% typical; under 1% best case
**Stablecoins win:** High-fee corridors, large txns, liquid off-ramps (Nigeria P2P, Philippines), B2B
**Stablecoins lose:** Small txns ($50-100), thin off-ramp liquidity, unbanked needing cash, strict KYC jurisdictions
**WU USDPT** stablecoin on Solana planned H1 2026; Digital Asset Network
**Scale:** Crypto remittances 3-5% of global flows (~$27-45B); USDT ~$703B/month mostly trading
