# Key Competitors — Loan Administration
Last updated: 26.4.9 | Reviews: R17

## Summary
The loan administration competitive landscape includes established agents (Alter Domus, GLAS, Kroll), specialized data/tech entrants (Hypercore, S&P DataXchange, S&P AmendX), and infrastructure platforms (Versana). Alter Domus is the scale incumbent; GLAS is the primary reference-class comparator for a de novo entrant given its trajectory from founding to institutional standing. The competitive window for a differentiated entrant is estimated at 12-18 months, XVERIFY confirmed.

## Strategic Paths for New Entrant
Three viable entry paths identified by reference class analysis [R17, 26.4.9]:
- Path A: De novo + experienced team (GLAS model) — highest P(success)
- Path B: Acquire + enhance — avoids cold start, team continuity risk
- Path C: De novo + tech-first without experienced team — no successful precedent, P = 0.15
- Key finding: success correlates with team experience, not entity formation method. Acquire-vs-build is a false dichotomy — the team is the variable. [R17, 26.4.9]
- Note: founding team industry experience level was identified as the largest single gap in the original analysis bundle — reference class evidence identifies it as the strongest predictor of breakeven timeline [R17, 26.4.9]
