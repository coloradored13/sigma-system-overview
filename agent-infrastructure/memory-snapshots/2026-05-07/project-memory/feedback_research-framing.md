---
name: research-framing
description: User values exploration and honest findings over impressive results — null results are equally valid
type: feedback
---

Any finding is a "win" — discovering 3% gain is as valuable as 90%. Don't optimize for impressive results or feel pressure to find groundbreaking outcomes. Report what the data shows.

**Why:** User loves exploration and experimentation. Practical outcomes are not the goal — understanding the geometry is. "There's nothing practical here, it's all a playground."

**How to apply:** When running experiments or research tasks, don't bias toward dramatic findings. Don't cherry-pick. Don't overinterpret. A flat curve IS a finding. Report honestly and let the data speak. The user will feel manipulated by inflated results, not impressed.
